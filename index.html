Yu-Gi-Oh! GENESYS Decklist Checker

Paste a plaintext decklist from EDOPro. This tool totals GENESYS points and checks for format legality using the latest local point list snapshot.

Notes

  • Point cap default is 100.
  • Any card not in the official GENESYS point list is treated as 0 points.
  • Link and Pendulum monsters are illegal in GENESYS.
  • This checker loads the bundled April 2026 point table and falls back to yugioh-card.com/en/genesys if needed.
